抱歉由於我們的技術尚不純熟，僅有能力處理嵌入mailto的功能，您可以嘗試將gmail設置為您的預設寄信軟體，或是點擊下方「我想直接看大家的e-mail」進行複製，造成您的不便，敬請見諒。
  • mac的設定
  • Windows預設寄信軟體不是google chrome時：
    控制台 → 程式集 → 預設程式 → 設定您的預設程式 → 電子郵件 → 選擇「Google Chome」
  • Windows預設寄信軟體雖是google chrome，但無法寄信時：
    開啟google chrome → 點擊右上方三個點並點選「設定」 → 滑動至頁面點選下方「進階」 → 於「隱私權與安全性」中點選「內容設定」 → 點選「處理常式」 → 勾選「允許網站要求成為通訊協定的預設處理常式」 → 設定後開啟gmail → 點選網址欄底部的雙箭號圖案（與處理常式的圖案相同) → 點選「允許」後，即可透過點選圖片進行寄信。
    若未出現處理常式的圖案時，可將瀏覽器關閉重啟，並重新點入gmail進行設定。
